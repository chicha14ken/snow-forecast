# ❄️ Snow Forecast — ゲレンデ選びアシスタント

長野・新潟の主要15スキー場のリアルタイム気象データを比較し、その日のベストゲレンデをランキング表示するWebアプリです。

## 特徴

- 🌨️ **パウダー重視モード** — 新雪量・積雪深を優先してランキング
- ☀️ **晴天重視モード** — 雲量・気温を優先して晴れゲレンデを提案
- ⚙️ **カスタムモード** — スライダーで重みを自由に調整
- 📡 **Open-Meteo API** — 無料・APIキー不要のリアルタイム気象データ
- 🗺️ **15ゲレンデ対応** — 白馬・志賀高原・野沢・新潟・妙高エリア

## 使い方

`index.html` をブラウザで開くだけで使えます。

## 対応ゲレンデ

| エリア | ゲレンデ |
|--------|---------|
| 白馬 | 八方尾根、白馬五竜、コルチナ、岩岳 |
| 志賀高原 | 横手山、焼額山、竜王 |
| 野沢・斑尾 | 野沢温泉、斑尾高原 |
| 新潟 | 苗場、かぐら、上越国際 |
| 妙高 | 杉ノ原、赤倉観光、池の平 |
